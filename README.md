# Scratch Premium Bag Selling Website

A full-stack e-commerce web application developed for selling bags.  
The project focuses on core e-commerce functionalities such as user authentication, product listing, order management, and secure online payments.

---

## Features
- User account creation and login system
- Dynamic product listing for bags
- Order management and checkout workflow
- Secure online payment integration using Razorpay
- Server-side rendered frontend for better control and simplicity

---

## Tech Stack
- Backend: Node.js, Express.js
- Database: MongoDB
- Frontend: EJS, CSS, JavaScript
- Payment Gateway: Razorpay
- Version Control: Git, GitHub

---

## Project Structure
config/ -> Database configuration
controllers/ -> Application logic
models/ -> Database models
routes/ -> Application routes
views/ -> EJS templates
public/ -> Static files (CSS, JS, images)
app.js -> Main server file
